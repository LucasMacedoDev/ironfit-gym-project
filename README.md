# 🏋️ IronFit Gym

Uma Landing Page moderna, responsiva e de alto impacto desenvolvida para a academia fictícia **IronFit Gym**.

Este projeto foi criado com o objetivo de demonstrar habilidades em desenvolvimento Front-End utilizando apenas tecnologias nativas da Web, priorizando código limpo, organização, responsividade e uma excelente experiência para o usuário.

# ⚡ Como executar

Clone o repositório:

```bash
git clone https://github.com/LucasMacedoDev/ironfit-gym-project.git
```

Acesse a pasta:

```bash
cd ironfit-gym-project
```

Abra o arquivo:

```
index.html
```

Não é necessário instalar dependências ou utilizar qualquer servidor.
